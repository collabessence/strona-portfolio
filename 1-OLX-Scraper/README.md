**Profesjonalny scraper ogłoszeń z OLX.pl z interfejsem graficznym**

## ✨ Funkcje

- ✅ Prosty interfejs graficzny (GUI)
- ✅ Scraping wielu stron jednocześnie
- ✅ Filtrowanie po słowach kluczowych
- ✅ Export do CSV i Excel

## 🚀 Instalacja

### Krok 1: Zainstaluj Python
Pobierz Python 3.8+ z: https://www.python.org/downloads/

### Krok 2: Zainstaluj biblioteki
Otwórz terminal w folderze projektu i uruchom:
```bash
pip install -r requirements.txt
```

### Krok 3: Uruchom aplikację
```bash
python olx_scraper.py
```

---

## 📖 Jak używać

1. **Podaj URL kategorii** z OLX (np. `https://www.olx.pl/elektronika/`)
2. **Wybierz liczbę stron** do przeszukania
3. **Opcjonalnie dodaj filtr** (np. "iPhone" aby znaleźć tylko ogłoszenia z iPhone)
4. **Kliknij "Rozpocznij Scraping"**
5. **Eksportuj wyniki** do CSV lub Excel
