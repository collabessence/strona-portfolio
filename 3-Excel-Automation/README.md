# 📊 Excel Automation Pro

**Profesjonalne narzędzie do automatyzacji pracy z plikami Excel**

## 💰 Wartość komercyjna: 200-500 zł

---

## ✨ Funkcje

### 🔗 Łączenie Plików
- Połącz wiele plików Excel w jeden
- Obsługa wielu arkuszy
- Automatyczne dodawanie kolumny źródła

### 🔍 Filtrowanie i Sortowanie
- Filtruj dane po dowolnej kolumnie
- Różne warunki (zawiera, równa się, większe/mniejsze)
- Sortowanie rosnące/malejące

### 📈 Analiza i Raporty
- Podstawowe statystyki (średnia, suma, min, max)
- Analiza unikalnych wartości
- Raport brakujących danych
- Grupowanie i sumowanie

### 🔄 Konwersja Formatów
- Excel → CSV
- CSV → Excel
- Wiele CSV → Excel (osobne arkusze)

```bash
pip install -r requirements.txt
```

### Krok 3: Uruchom aplikację


## 📖 Jak używać

### Łączenie Plików
1. Kliknij "Dodaj Pliki" i wybierz pliki Excel
2. Wybierz opcje łączenia
3. Kliknij "Połącz Pliki"
4. Zapisz wynik

### Filtrowanie
1. Wybierz plik do filtrowania
2. Wybierz kolumnę i warunek
3. Wpisz wartość do filtrowania
4. Opcjonalnie: wybierz sortowanie
5. Kliknij "Zastosuj Filtry"

### Analiza
1. Wybierz plik do analizy
2. Wybierz rodzaj analizy
3. Kliknij "Generuj Raport"
4. Zapisz raport
