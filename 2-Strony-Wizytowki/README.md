## 🎨 Customizacja


```css
/* Na górze CSS dodaj import: */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600;700&display=swap');

/* Zmień font-family: */
body {
    font-family: 'Poppins', sans-serif;
```

## 🐛 Troubleshooting

### Problem: Strona nie wyświetla się poprawnie

**Rozwiązanie:**
- Sprawdź ścieżki do plików CSS/JS
- Upewnij się że wszystkie pliki są wgrane
- Otwórz Dev Console (F12) i sprawdź błędy

### Problem: Formularz nie wysyła

**Rozwiązanie:**
- Skonfiguruj backend (Formspree, EmailJS)
- Lub zmień `action` w `<form>` na swój endpoint
- Sprawdź czy JavaScript jest załadowany

### Problem: Mapa Google nie działa

**Rozwiązanie:**
- Zmień `src` iframe na swój embed link z Google Maps
- Upewnij się że link zawiera `embed/`

### Problem: WhatsApp button nie działa

**Rozwiązanie:**
- Zmień numer w `href="https://wa.me/48123456789"`
- Usuń spacje z numeru
- Dodaj +48 przed numerem
